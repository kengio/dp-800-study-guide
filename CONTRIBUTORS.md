# Contributors

This guide exists because someone open-sourced their exam notes — and grows because *you* improve it. Every PR that lands gets the author attributed here.

## How to get added

When your PR merges, either:

1. **Open a follow-up PR** adding yourself to the list below, **or**
2. **Mention it in the original PR** and a maintainer will add you when merging

If you'd prefer not to be listed publicly, no problem — just say so in your PR description and we'll skip the entry while still merging the change.

## Attribution format

Each entry uses this shape:

```markdown
- **[Display name](https://github.com/<handle>)** — short description of contributions (e.g., "RAG topic refresh", "Always Encrypted clarifications", "mock-exam question additions", "broken-link fixes")
```

Keep descriptions ≤ 100 characters. Authors who contribute across many areas can summarise (e.g., "ongoing maintenance").
