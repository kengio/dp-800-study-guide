# DP-800: Developing AI-Enabled Database Solutions

Study guide for the Microsoft DP-800 certification exam.

## About the Exam

DP-800 tests expertise in designing and developing AI-enabled database solutions across Microsoft SQL platforms — Microsoft SQL Server, Azure SQL, and SQL databases in Microsoft Fabric.

| Detail            | Information                                   |
| ----------------- | --------------------------------------------- |
| **Exam**          | DP-800                                                   |
| **Full Name**     | Developing AI-Enabled Database Solutions                 |
| **Passing Score** | 700 / 1000                                               |
| **Renewal**       | Annual (free online assessment on Microsoft Learn)       |
| **Languages**     | T-SQL                                                    |
| **Platforms**     | SQL Server, Azure SQL, SQL databases in Microsoft Fabric |

## Exam Domains

| Domain | Weight |
| ------ | ------ |
| Design and develop database solutions | 35–40% |
| Secure, optimize, and deploy database solutions | 35–40% |
| Implement AI capabilities in database solutions | 25–30% |
